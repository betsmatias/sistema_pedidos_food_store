# Sistema de Pedidos (Food Store) 

## Descripción
Proyecto para la materia Programación 2. Consiste en un sistema de gestión de pedidos mediante consola desarrollado en Java. 
Aplica los pilares de la Programación Orientada a Objetos, arquitecturas basadas en capas 
(Servicios y Entidades), manejo de excepciones personalizadas y persistencia de datos en memoria.

## Cómo ejecutar el proyecto
1. Clonar el repositorio en tu máquina local.
2. Abrir el proyecto en un IDE compatible con Java (NetBeans, IntelliJ IDEA, Eclipse).
3. Asegurarse de tener configurado el JDK 17 o superior.
4. Navegar hasta la clase principal: `src/integrado/prog2/Main.java`.
5. Ejecutar la clase `Main`.
6. Interactuar con el sistema a través del menú de la consola ingresando los números de las opciones.
